# ae2f::Core
- Rel-v6.25.19

> This project has been migrated to [codeberg.org](https://codeberg.org/ae2f/Core).

> Integration between C and C++ via CMake.

> The smallest core utils for ae2f-like library.  
> Contains no source, just headers. (interface)

- C         >= 90
- C++       >= 14   [Optional]
